## Training Path

### For Green Engineers (No Oilfield Experience)

**Week 1:**
1. Read 00_QUICK_START.md
2. Read 05_GLOSSARY.md (drilling terms section)
3. Shadow experienced driller using system

**Week 2:**
1. Read 02_CONCEPTS_GUIDE.md
2. Practice identifying regimes during normal drilling
3. Ask questions about anything unclear

**Week 3:**
1. Read 01_FIELD_OPERATORS_MANUAL.md
2. Practice data entry (formations, BHA)
3. Do a supervised shift with primary responsibility for system

**Ongoing:**
- Reference 04_TROUBLESHOOTING_FAQ.md as needed
- Review 05_GLOSSARY.md for new terms

### For Experienced Hands New to the System

**Day 1:**
1. Read 00_QUICK_START.md
2. Skim 02_CONCEPTS_GUIDE.md

**Day 2:**
1. Read 01_FIELD_OPERATORS_MANUAL.md
2. Use system during shift, noting questions

**Day 3:**
1. Re-read any sections that were unclear
2. Read 04_TROUBLESHOOTING_FAQ.md

**Ongoing:**
- Reference as needed

### For Supervisors

**Before deployment:**
1. Read 03_SUPERVISORS_REFERENCE.md
2. Understand ROI and training requirements

**During deployment:**
1. Ensure crew has appropriate documents
2. Review 04_TROUBLESHOOTING_FAQ.md for support escalation
